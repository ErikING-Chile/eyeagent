# eyeagent

## Proyecto

Este repositorio contiene **Arche** - un agente de IA de código abierto para desarrollo de software, con integración opcional de Telegram.

### Características
- 🤖 **Agente de IA** para programación
- 📱 **Bot de Telegram** para conversar con el agente
- 💾 **Persistencia de memoria** entre sesiones

## Clonado con submodulos
Recomendado:

```bash
git clone --recurse-submodules <url-del-repo>
```

Si ya clonaste sin submodulos:

```bash
git submodule update --init --recursive
```
